## 使用手册
1. 首先收到query-1.0.0-SNAPSHOT.jar
2. 确保8080端口未被占用
3. 确保本地有Jre1.8环境
4. 命令行进入jar文件的目录下
5. 执行 java -jar query-1.0.0-SNAPSHTO.jar 命令行
6. 项目启动完成
